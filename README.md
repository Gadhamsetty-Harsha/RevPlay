# 🎧 P1 – RevPlay

## 📌 Overview
RevPlay is a console-based music streaming application. You explore and manage songs, artists, playlists, albums. You search content, mark favorites, manage playlists, and simulate playback. Artists create profiles and publish music. The application uses a modular, layered design and supports future web or mobile expansion.

## 👤 User Features
### Account
- 📝 Register and create an account
- 🔐 Secure login
- 🔁 Change password
- ❓ Recover forgotten password
  
### Discovery
- 🔍 Search songs, artists, playlists, albums
- 📂 Browse by genre, artist, album

### Engagement
- ❤️ Mark songs as favorites
- 📃 View favorite songs
- 🕒 View recently played tracks
- 📊 View listening history

### Playback Simulation
- ▶️ Play
- ⏸️ Pause
- ⏭️ Skip
- 🔁 Repeat
Text-based simulation

### Playlist Management
- 📀 Create playlists. Public or private
- ➕ Add songs
- ➖ Remove songs
- ✏️ Update name, description, privacy
- 🗑️ Delete owned playlists
- 🌍 View public playlists from other users

## 🎤 Artist Features
### Account
- 🧑‍🎤 Register as an artist

- 📝 Create and manage artist profile

### Content Management
- 🎶 Upload songs with metadata

### Insights

- 📊 Track song statistics
